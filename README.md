# Hybrid-PINN-LLM-Framework-for-Supercritical-Aeroelastic-Flutter-Control
Hybrid 2-DOF wing control integrates PINNs for low-level execution with an LLM supervisor for strategy adaptation. By targeting supercritical flutter at (Vf), it arrests catastrophic failure where static controllers fail. This physics-aware, adaptive architecture ensures structural stability beyond theoretical limits.
